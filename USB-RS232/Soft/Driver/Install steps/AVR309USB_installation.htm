AVR309:USB to UART device installation

 

Hardware installation

Your computer must be running Microsoft Windows 98, Windows ME, Windows 2000 or Windows XP.
Under Windows 98
Under Windows XP

Note: The pictures displayed bellow in each step are for AVR309:USB to UART device Advanced version (first picture) and for Simple version (second picture).

Installation steps under Windows 98:

  1. Your computer must be switched on and running Microsoft Windows.
  2. Connect AVR309:USB to UART device based device to USB port on your computer.
  3. "New hardware found" window appears (with "AVR309:USB to UART ..." name).
  4. Window "Add New Hardware Wizard" will be opened.
  5. Click "Next":
  6. Choose "Search for the best driver for your device (Recommended)" and click "Next":
     
  7. Check "Specify location" and "Browse" to the folder where are driver files stored (CD or download folder):

    And then click "Next".
  8. In the next window (with founded AVR309:USB to UART protocol converter ...) click "Next":

    Windows then will install necessary driver files and makes system changes. Rebuild of Windows driver database may appears.
  9. Installation is complete. Click "Finish" in the last window:
  10. Check if the device "AVR309:USB to UART protocol converter ..." is present in "Start - Settings -Control Panel - System - Device Manager - Universal Serial Bus controllers":
  11. Device was successfully installed.

Installation steps under Windows XP:

  1. Your computer must be switched on and running Microsoft Windows.
  2. Connect AVR309:USB to UART  ... based device to USB port on your computer and wait.
  3. "Found new hardware" window appears (with "AVR309:USB to UART ..." name):
  4. Window "Found New Hardware Wizard" will be opened.
    Choose "Install from a list or specify location (Advanced)" and click "Next":
  5. Check "Include this location in the search" and "Browse" to the folder where are driver files stored (CD or download folder):

    And then click "Next".
  6. The next window (with device AVR309:USB to UART protocol converter ...) appears:

    If  warning window "Hardware installation" appears, choose "Continue Anyway":
  7. Windows then will install necessary driver files and makes system changes:
  8. Installation is complete. Click "Finish":
  9. After successful installation the next small window with confirmation of installation appears:
  10. Check if the device "AVR309:USB to UART protocol converter ..." is present in "Start - Settings - Control Panel - System - Device Manager - Universal Serial Bus controllers":
  11. Device was successfully installed.
